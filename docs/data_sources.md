# Data Sources and Privacy Notes

## Public Data Used

- California Department of Public Health, Licensed and Certified Healthcare Facility Listing.
- Local file used by this project: `data/california_healthcare_facilities_raw.csv`.

## Synthetic Data Used

The county-level population, income, poverty, uninsured, rural/urban category, and ER visit-rate fields in this portfolio project are synthetic demonstration indicators. They are designed to resemble realistic public-health analytics fields but should not be used for operational decisions.

## Privacy

This project does not use patient-level records, claims, medical chart data, appointment data, addresses of patients, or protected health information. The unit of analysis is county-level community risk.
